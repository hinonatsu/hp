# Manufacturing Website Before After Demo

架空のゴム製品製造会社「東霞ラバー工業株式会社」を使った、営業用のBefore/Afterデモです。
従来型の情報中心サイトと、取引先・求職者に伝わる順番へ整理した改善後サイトを比較できます。

## 目的

技術系・現場系の中小企業に対して、現在のホームページを否定するのではなく、すでに持っている技術・設備・品質体制・人の魅力をWeb上で整理すると印象がどう変わるかを体感してもらうためのデモです。

このデモは架空企業を用いた営業用サンプルです。実在の企業・団体とは関係ありません。

## 起動方法

```bash
npm install
npm run dev
```

起動後、表示されたローカルURLをブラウザで開きます。
初期表示は `#compare` です。`#before`、`#after` でも直接開けます。

## build方法

```bash
npm run build
```

TypeScriptの型チェック後、Viteで本番ビルドを作成します。

## 営業で見せるときの説明例

「こちらは架空のゴム製造会社を使ったBefore/Afterデモです。従来型の情報中心サイトと、採用・信用確認に向けて情報を整理した改善後サイトを比較できます。御社の現在のサイトを否定する意図ではなく、すでにお持ちの技術・設備・品質体制を、取引先や求職者に伝わる順番へ整理するとどう見えるかを示すためのサンプルです。」

## 注意

- 実在企業の公式サイト、ロゴ、写真、スクリーンショット、文章は使用していません。
- 画像素材、外部フォント、外部UIライブラリ、Tailwind CSSは使用していません。
- 問い合わせボタンは実送信せず、デモ用の注意メッセージのみ表示します。
- `index.html` には `noindex,nofollow` を設定しています。
